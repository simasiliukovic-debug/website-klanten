\# Website Klanten



\## Beschrijving

Webapplicatie om NAW-gegevens van klanten te beheren (CRUD).



\## Bestanden

\- `db.php` → databaseverbinding

\- `index.php` → overzicht van alle klanten

\- `toevoegen.php` → formulier om klant toe te voegen

\- `insert.php` → verwerkt toevoegen

\- `edit.php` → formulier om klant te bewerken

\- `update.php` → verwerkt wijzigen

\- `delete.php` → verwijdert klant

\- `style.css` → styling



\## Installatie / Starten

1\. Installeer XAMPP en start \*\*Apache + MySQL\*\*  

2\. Plaats alle bestanden in `C:\\xampp\\htdocs\\website`  

3\. Maak database `klanten\_db` aan (zie SQL in `db.php`)  

4\. Open browser: `http://localhost/website/index.php`
